# Backgammon_Kartenspiel
## Allgemein
Kann Backgammon als Kartenspiel funktionieren? Ich habe mich mit diesem Projeket dieser Frage genähert und versucht eine Print&Play Variante dieses Spiels zu getalteten.
## Wie ist das Repositorie aufgebaut? 
### Ordner "DRUCKDATEI"
[Hier](https://github.com/ivohartwig/Backgammon_Kartenspiel/tree/main/DRUCKDATEI) findet ihr die Illustrator Datei und im Unterordner "PDF" die fertige PDF zum downloaden und drucken. 
### Ordner "DATEIEN"
[Hier](https://github.com/ivohartwig/Backgammon_Kartenspiel/tree/main/DATEIEN) findet ihr alle anderen Dateien der einzelnen Elemente der Druckdatei. Im Unterordner "PDF" findet ihr alle dazugehörigen PDF´s. 
### Ordner "WÜRFEL_APP"
[Hier](https://github.com/ivohartwig/Backgammon_Kartenspiel/tree/main/WÜRFEL_APP) findet ihr einen Download für eine Würfel App.
